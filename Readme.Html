📦 Qasr-e-Zaiqa Inventory Panel

This panel is a lightweight, browser-based inventory tracking tool designed for the daily operations of Qasr-e-Zaiqa Restaurant. It allows staff to manually enter real-time stock usage and helps owners view updated inventory status from anywhere.

🔧 Key Features

  • Live manual stock updates
  • Editable quantity and user fields
  • Separate tables for Stock IN and OUT
  • Section hide/collapse buttons for easy viewing
  • Export to PDF / Print option (coming soon)

📁 Files Included

  • index.html – Main inventory system interface
  • README.html – This documentation file

📌 Usage

  1. Open index.html in any browser
  2. Enter the current quantity and who used the item
  3. Update manually as stock is used or received
  4. Export or print the report (feature coming soon)

🚀 Future Plans

  • Role-based login (Admin / Chef)
  • Firebase or local DB support
  • Daily closing summaries
  • Low stock alerts & notifications
